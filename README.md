# ARCA Site

Site vitrine do ARCA, aplicativo desktop da SIAPESQ para monitoramento inteligente de espécies.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
```

## Download do instalador

Os botões de download apontam para:

```text
/downloads/ARCA-Setup-0.1.0.exe
```

No projeto, coloque o instalador em `public/downloads/ARCA-Setup-0.1.0.exe`.
